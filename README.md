# Delhi Property Calculator

A web-based application built using Python and Streamlit to assist with property-related calculations specific to Delhi.

## Features
- Residential property calculations
- Commercial property calculations
- Colony-based and circle rate logic
- Basic admin-side handling for data management

## Tech Stack
- Python
- Streamlit
- CSV-based data handling

## How to Run
1. Install dependencies:
   pip install -r requirements.txt
2. Run the application:
   streamlit run app.py

## Purpose
This project was developed to apply Python programming skills to a real-world administrative and property-related use case.

## Status
Working basic version used for practical learning.
